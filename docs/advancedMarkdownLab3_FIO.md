Alert-блок

>[!TIP]
>**Совет по использованию:** Для экранирования специальных символов в Markdown используйте обратный слэш \. Например, чтобы написать *курсив* без форматирования, напишите \*курсив\*.

### Escape-символы

- Заголовок без форматирования: \# Это не заголовок
- Звёздочка как текст: \*Просто звёздочка\*

### HTML в Markdown
```HTML
<p style="background-color: #fff9fa; padding: 15px; border-radius: 5px;">
Этот параграф оформлен с помощью <strong>HTML-тега</strong> внутри Markdown-документа.
</p>
```


[^1]: Синтаксис Markdown поддерживается на GitHub, GitLab, в большинстве редакторов кода и систем документирования. ➕
